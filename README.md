📌 Project Overview

PDF Vocabulary Reader is an AI-powered web application that allows users to upload PDF files and instantly view word meanings while reading.
It helps students improve vocabulary and comprehension by making difficult words understandable in real-time.

🚀 Key Features

📄 Upload any PDF file

🔍 Click on words to get instant meaning

🤖 AI-based vocabulary assistance

⚡ Fast and lightweight Streamlit interface

🌐 Deployed on Streamlit Cloud

📱 Simple and user-friendly UI


🧠 Problem It Solves

While reading PDFs (books, notes, research papers), students often struggle with difficult vocabulary and constantly switch apps to check meanings.
This project solves that problem by providing instant word meaning inside the document reader itself, improving focus and learning efficiency.

🏗️ How It Works (System Workflow)

User uploads a PDF file

The app extracts text from the PDF

User interacts with words in the document

The system fetches and displays the meaning of selected words

Output is shown instantly on the web interface
